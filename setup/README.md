# Mailu-OIDC Setup Utility

> [!CAUTION]
> This utility is still in development and may not work as expected.

This is Heviat's [SvelteKit](https://kit.svelte.dev/) based setup utility for
[Mailu-OIDC](https://github.com/heviat/Mailu-OIDC). It is a more user-friendly
alternative to the original setup utility.

> [!TIP]
> You can also use this utility to setup `Mailu-OIDC` with OIDC disabled, or
> the original `Mailu` without OIDC.
